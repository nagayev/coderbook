Объект Math

Объект Math используется для выполнения математических операций и извлечения значений констант.

Дополнительную информацию о использовании данного объекта можно найти в нашем JavaScript учебнике.

Обратите внимание: чтобы воспользоваться свойствами и методами математического объекта его не нужно предварительно создавать.

		// Отобразим значения числа Пи
		document.write(Math.PI);
		
		// Отобразим значения натурального логарифма 10 
		document.write(Math.LN10);

Свойства объекта Math

Свойство Описание
E Возвращает число Эйлера (примерно 2,71).
LN2 Возвращает значения натурального логарифма (с основанием E) двух (примерно 0,7).
LN10 Возвращает значение натурального логарифма (с основанием E) десяти (примерно 2,3).
LOG2E Возвращает значение логарифма E с основанием 2 (примерно 1,4).
LOG10E Возвращает значение логарифма E с основанием 10 (примерно 0,4).
PI Возвращает значение числа Пи (примерно 3,14).
SQRT1_2 Возвращает квадратный корень 1/2 (примерно 0,7).
SQRT2 Возвращает квадратный корень 2 (примерно 1,4).

Методы объекта Math

Обратите внимание:Тригонометрические функции возвращают значения в радианах

Методы Описание
abs(x) Возвращает абсолютное значения (модуль) числа x.
acos(x) Возвращает арккосинус числа x
acosh(x) * Возвращает гиперболический арккосинус числа x
asin(x) Возвращает арксинус числа x
asinh(x) * Возвращает гиперболический арксинус числа x
atan(x) Возвращает арктангенс числа x как численное значение между -PI/2 и PI/2.
atanh(x) * Возвращает гиперболический арктангенс числа x.
atan2(y,x) * Возвращает арктангенс от частного своих аргументов.
cbrt(x) * Возвращает кубический корень числа x.
ceil(x) Округляет значение x до первого большего целого числа.
cos(x) Возвращает косинус числа x.
exp(x) Возвращает значение E в степени x.
floor(x) Округляет значение x до первого меньшего целого числа.
log(x) Возвращает натуральный логарифм (с основанием E) х.
log2(x) Возвращает логарифм (с основанием 2) х.
log10(x) Возвращает логарифм (с основанием 10) х.
max(x1,x2,...xn) Возвращает большее из чисел x1,x2,...xn.
min(x1,x2,...xn) Возвращает меньшее из чисел x1,x2,...xn.
pow(x,y) Возводит x в степень y и возвращает результат. Аналогично x**y
random() Возвращает случайное число между 0 и 1 (например 0.6230522912910803).
round(x) Округляет значение x до ближайшего целого числа.
sin(x) Возвращает синус числа x (число x задается в радианах).
sqrt(x) Возвращает квадратный корень x.
tan(x) Возвращает тангенс угла.

* - данная функция добавлена в стандарте ES2015 (ES6) и может быть недоступна в старых браузерах, таких как Internet Explorer.
